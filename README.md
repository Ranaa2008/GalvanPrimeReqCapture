## 🔐 Permissions System

### Available Permissions
- `view users`, `create users`, `edit users`, `delete users`
- `view roles`, `create roles`, `edit roles`, `delete roles`
- `view permissions`, `create permissions`, `edit permissions`, `delete permissions`
- `view admin dashboard`

### Usage in Code

**Protect Routes:**
```php
// Protect with role
Route::middleware(['auth', 'role:admin'])->group(function () {
    // Your routes
});

// Protect with permission
Route::middleware(['auth', 'permission:edit users'])->group(function () {
    // Your routes
});
```

**Check in Controllers:**
```php
if (auth()->user()->hasRole('admin')) {
    // Admin specific code
}

if (auth()->user()->can('edit users')) {
    // Permission specific code
}
```

**Check in Views:**
```blade
@can('edit users')
    <button>Edit User</button>
@endcan

@role('admin')
    <a href="/admin">Admin Panel</a>
@endrole
```

## 🎨 Customization

### Adding New Permissions

**Via Admin Panel:**
1. Login as admin
2. Navigate to Permissions
3. Click "Create New Permission"
4. Enter permission name (e.g., "view reports")

**Via Seeder:**
Edit `database/seeders/RolePermissionSeeder.php`:
```php
$permissions = [
    'your new permission',
    // ... other permissions
];
```

### Adding New Roles

**Via Admin Panel:**
1. Login as admin
2. Navigate to Roles
3. Click "Create New Role"
4. Select permissions to assign
